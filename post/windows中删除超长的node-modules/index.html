windows中删除超长的node modules

11 Oct 2016, 13:41
杂项
Node.js

在Node.js开发中，依赖第三方项目，会产生比较深的文件目录，Windows下无法简单删除

解决方法

  • 新建空白目录，如F:\git\temp
  • 管理员方式打开命令行窗口
  • 输入robocopy F:\git\temp F:\git\test\node_modules /purge
  • 回车，搞定 更多关于robocopy的说明请看 robocopy
在win10中运行zsh或者其他shell
